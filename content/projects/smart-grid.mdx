# Smart Grid Monitoring System

A comprehensive IoT-based power grid monitoring system developed as a senior design project.

## Overview

This project implements a real-time monitoring solution for power distribution networks, featuring custom hardware for data acquisition and a web-based dashboard for visualization.

## Features

- Real-time voltage and current monitoring
- Power quality analysis (THD, power factor)
- LoRa-based wireless communication
- Cloud data storage and analytics
- Alert system for anomalies

## Technical Stack

### Hardware
- Custom PCB with STM32 microcontroller
- ADE7753 energy measurement IC
- SX1276 LoRa transceiver
- Industrial-grade sensors

### Software
- Firmware: C with FreeRTOS
- Backend: Node.js with Express
- Frontend: React with Chart.js
- Database: PostgreSQL with TimescaleDB
